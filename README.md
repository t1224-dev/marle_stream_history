# マールの軌跡アプリ

マール・アストレアさんの活動記録をまとめたファン向けアプリケーションです。

## 主な機能

### カレンダー表示
- 月別/週別/日別で活動日を確認
- 日付選択でその日の配信動画を表示

### 動画検索
- キーワード検索
- 日付範囲やタグでフィルタリング
- グリッド/リスト表示切り替え

### 動画詳細
- サムネイル・タイトル・説明文表示
- お気に入り登録機能
- 共有機能（SNSなどへ）

### お気に入り管理
- 気に入った動画を保存
- お気に入り一覧表示

### その他機能
- 表示レイアウトカスタマイズ
- お問い合わせ機能

## 技術仕様
- 開発フレームワーク: Flutter
- 状態管理: Riverpod
- ローカルDB: SQLite
- デザイン: Material Design 3
